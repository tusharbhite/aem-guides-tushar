Sightly: Block Statements

1. data-sly-test

Validates and conditionally removes the host element and it's content. Returned 'False' removes the element; Returned value 'true' retains the element

Statement:  

<p data-sly-test="${'${currentPage.title==dev}'}"> Current Page Title is dev </p>

HTML Output:  

<p> Current Page Title is dev </p>

HTML Render:  

Current Page Title is dev

Statement:  

<p data-sly-test="${'${!currentPage.title==dev}'}"> Current Page Title is not dev</p>

HTML Output:  

<p> Current Page Title is not dev </p>

Rendered Output:  

Current Page Title is not dev

2. data-sly-use

Used to initialize a Java class/Sling Model or JavaScript object. Parameters can be passed to the initialization using options

Statement:  

<p data-sly-use.componentModel="com.adobe.cq.wcm.core.components.models.Component"> ${'${ componentModel.id }'} </p>

HTML Output:  

<p> sightly-ad2823e02b </p>

HTML Render:  

${ componentModel.id }

3. data-sly-unwrap

Trims the html element from the generated markup.

Statement:  

<tag-to-be-removed data-sly-unwrap><p>Parent Tag Removed Successfully</p></tag-to-be-removed>

HTML Output:  

<p> Parent Tag Removed Successfully </p>

HTML Render:  

Parent Tag Removed Successfully

4. data-sly-text

Inserts the content of its host element with the specified text by replacing it. Basically it will make current element empty (It will remove all child tags or texts anything) and insert value of expression inside current element.

Statement:  

<tag-to-be-kept data-sly-text="${'${currentPage.title}'}"><p>Parent Tag kept Successfully</p></tag-to-be-kept>

HTML Output:  

<tag-to-be-kept> dev </tag-to-be-kept>

HTML Render:  

Parent Tag Kept Successfully

5. data-sly-attribute

Helps to add attributes to the host element.

Statement:  

<p title="Page Title" data-sly-attribute.title="${'${currentPage.title}'}" > Title </p>

HTML Output:  

<p title="dev" > Title </p>

HTML Render:  

Title

6. data-sly-element

Used to replaces the element name of the host element of html tags.

Statement:  

<p-old data-sly-element="${'${p}'}" > Title </p-old>

HTML Output:  

<p > Title </p>

HTML Render:  

Text
